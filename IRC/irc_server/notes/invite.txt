

/INVITE : invite a user to a channel. If channel is invite only [+i], user
		  who's doing the invite must be recognized as a channel operator

usage : /INVITE [user] <channel>

/example:

   :Angel INVITE Wiz #Dust         ; User Angel inviting WiZ to channel
                                   #Dust

   INVITE Wiz #Twilight_Zone       ; Command to invite WiZ to
                                   #Twilight_zone
